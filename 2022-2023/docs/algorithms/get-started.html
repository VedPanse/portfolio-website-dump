Written By Ved Panse Last Modified: 30 June 2021

In this section of the documentation, you will find information that might come in handy (not necessarily now, but in the future). This documentation, though has been made for people without any technical knowledge, needs some information that the documentation uses. This section does just that. It provides you an insight of the terms that we will be using in the documentation. Feel free to visit this page again if you get stuck or cannot figure out the meaning of any word in the docs.

Let's get started!

How to make use of this documentation?

This documentation covers just a tiny speck of a vast topic: Algorithms. The main purpose of this documentation is to show you a sneak-peak of algorithms. Though this documentation will not take you into the nitty-gritty of the topic, we will be touching some of the common problems and applications of the algorithms you will use.

This documentation covers a discussion on algorithms which are both: solved and unsolved. We will be diving into this topic later in the documentation.

There are several code terminals used in this documentation. If, by chance, the entire code is not visible in the terminal, and gets cut off due to small size, try sliding along the terminal to view the entire code.
