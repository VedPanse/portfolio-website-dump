Histograms

Histograms represent the distribution of data. Just like other graphics, they are vital to predicting many outcomes.

Let us start by importing the modules we will need.

import plotly.express as px
import numpy as np

In this example, we will not be importing any dataset. Instead, we will be creating our own using NumPy. This dataset represents the outcomes when two dice are rolled simultaneously 5000 times. The data is random so the results will be different most of the time.

dice_1 = np.random.randint(1, 7, 5000)   # Dice will be rolled 5000 times
dice_2 = np.random.randint(1, 7, 5000 )

dice_sum = dice_2 + dice_1

Let us plot this data on a histogram by using plotly.express.

fig = px.histogram(dice_sum, nbins=11, labels={"value": "Dice Roll" },
                   title="5000 Dice rolls", marginal="violin",
                   color_discrete_sequence=['green'])  # nbins: means the number of experiments to be done

Let us configure some of the properties of the above graph.

fig.update_layout(
    xaxis_title_text="Dice rolls",
    yaxis_title_text="Dice sum",
    bargap=0.2,
    showlegend=False
)
fig.show()
