Box Plots

Written By Ved Panse Last Modified: 19 May 2022

Box plots are used to represent the spread and skewness of data. Let us start plotting this important figure by importing the required modules.



Importing Libraries

import plotly.express as px
import plotly.graph_objects as go

We will use df_tips as our data frame. We will import this data frame from plotly. The tips data frame presents tips received by employees at an imaginary restaurant.

Let the x-axis and y-axis represent "sex" and "tip" respectively.

fig = px.box(df_tips, x="sex", y="tip", points="all")
fig. show()

Here, the points are set to all so that all the points get displayed. This is what your plot should look like.
