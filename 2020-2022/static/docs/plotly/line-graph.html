Line Graphs

Written By Ved Panse Last Modified: 18 May 2022

2-Dimensional

2-D line plots are generally to visualize simple datasets. The data is represented by a line or more. Line plots frequently help evaluate stock markets. Let us create one using python.

We have to start by uploading a dataset first. Fortunately, plotly has some inbuilt sets for us.

df_stocks = px.data.stocks()

Let's plot the graph.

graph = px.line(df_stocks, x="date", y="GOOG", labels={"x": "Date", "y": "Price"})
# labels are the titles for respective axes.

graph.show()

On running the program, you will find a similar graph. Specific data gets displayed by hovering over a point, as shown in the image above. Here, we can precisely monitor the price of stocks on a given date.

3-Dimensional

3-D line graphs also help to visualize simple datasets. The only difference is in their visualization. As the name states, 3-D line graphs represent the data in a 3-D way. Seaborn is another Python data-visualization library.

import seaborn as sns
import plotly.express as px

flights = sns.load_dataset("flights")
fig = px.line_3d(flights, x= "year", y="month", z="passengers", color="year")
fig. show()
