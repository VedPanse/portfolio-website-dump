Time Series Algorithms

Written By Ved Panse Last Modified: 20 May 2022

A time-series algorithm is an algorithm that runs till infinity and has periodicity.

ARIMA

ARIMA stands for Autoregressive Integrated Moving Average. It generalizes a time series, thus simplifying the data and helping to make a prediction.

Image from https://miro.medium.com/max/789/1*Z7QvavK7s_8JAqCRzkMUvA.png

The above image shows how the original and the general graphs differ.

SARIMA

The SARIMA algorithm is simply an extension of the ARIMA algorithm. The difference is the 'S' that stands for 'Seasonal.' The SARIMA enables ARIMA to handle data with seasonality.

What is seasonality? Let us see below.

Seasonality

Seasonality means periodicity. A seasonal event is an event that is certain to happen at a given time.
For example, a person's birthday is sure to come after one year on the same date. A sudden rise in profits is observed by resorts/hotels in the summer as people go out for vacations. Such kinds of events make up seasonal events.

Image from https://miro.medium.com/max/960/1*1cxk8WImBj3pWOyVWQHwdg.png

Here, we can see that the data is recurring.

Back to SARIMA

ARIMA, as we read, does not handle seasonality well. Thus, SARIMA came up to fill in this flaw.

TBATS

TBATS has a complex long-form: Trigonometric seasonality, Box-cox transformation, ARMA errors, Trend components, and Seasonal components. It first carries out exponential smoothing on the data. Next, it uses smooth data to make predictions. It is different from ARIMA as it can also recognize patterns in complex graphs.
