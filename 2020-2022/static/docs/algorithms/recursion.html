What are recursive programs?

Recursive programs are those which keep calling themselves until they are told to stop. If not stopped, they can perform infinite loops. 



def greet(name):
        print(f"hello, {name}!")
        greet2(name)
        print("getting ready to say bye...")


def greet2(name):
        print(f"How are you {name}?")

        def bye():
            print("ok bye!")
        bye()

        greet("Ved")


def fact(x): #fact -> factorial
        if x == 1:
            return 1
        else:
            return x * fact(x-1)

print(fact(12))
