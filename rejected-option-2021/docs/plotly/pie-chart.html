Simple Pie Charts

The pie chart is a circular representation of data. It can represent the data in proportional format.

Let us plot one in Python. We will import our usual libraries and get started with them.

import plotly.express as px

Let us import Asia's 2007 data using the gapminder() module provided by plotly. We will call this data df_asia.

df_asia = px.data.gapminder().query("continent == 'Asia' and year == 2007" )

Now, we will plot the pie chart. The color_discrete_sequence provides a theme color for the pie chart. Refer to this link this link for the complete list of themes. We will be using the Rainbow theme. A pie chart can represent only one variable. This variable is called value. Let's plot the pie chart of the 'population' value.

pie = px.pie(df_asia, values="pop", names="country", title="Population by countries in Asia",
             color_discrete_sequence=px.colors.sequential.Rainbow)
pie. show()

Your pie chart should look somewhat similar to this one.

Pie Chart With Additional Features

You can add many features to your pie chart using Plotly's graph objects. Let us import the required module.

import plotly.graph_objects as go

Let us create an array of colors that we could use on the pie chart. This would do nothing but simplify the code. Let us create a figure representing the number of students in a school who prefer a particular sport.

colors = ['blue', 'green', 'black', 'purple', 'red', 'brown' ]

fig = go.Figure(data=[go.Pie(labels=[ "Volleyball", "Cricket", "Football", "Hockey", "Badminton", "Tennis"],
                             values=[ 110, 20, 100, 12, 23, 12]) ])

Here, the labels refer to the variables that are represented by the pie chart. The values refer to the corresponding values of labels. Let us update some features of this figure.

fig.update_traces(hoverinfo="label+percent", textfont_size=20, textinfo='label+percent',
                  marker=dict(
                      colors=colors,
                      line=dict(color='#ffffff', width= 2)
                  ))
fig. show()

Here, hoverinfo means the information shown when the mouse pointer hovers over the pie chart. The marker refers to the graphic (borders). The colors are set to the colors array. The borderline is set to #ffffff (white) and the width is 2. Also, note that you can write the color in RGB form. You'll find your graph to be somewhat similar to the one shown here.
