More About Supervised Learning

Written By Ved Panse Last Modified: 20 May 2022

Supervised learning looks at the previous data and makes a prediction. Generally, predictions use two methods:

  1. Regression
  2. Classification

Regression

Regression is somewhat similar to creating a line of best fit. The line of best fit is the line that best represents the data. The values on this line are approximately equal to the provided values. This line is expanded to predict an outcome.

Image from https://upload.wikimedia.org/wikipedia/commons/b/be/Normdist_regression.png

If we provide an x-axis coordinate to this algorithm, it will return the corresponding y-axis coordinates on the line of best fit.

Classification

Classification deals with categorizing data into groups by drawing a line in between their graphs.

Image from https://static.javatpoint.com/tutorial/machine-learning/images/classification-algorithm-in-machine-learning.png

The data is plotted on the graph considering only two factors: their x-axis and y-axis representations. The data is classified based on their respective locations relative to the line. In other words, if a line divides a dataset into two, the data points below the line are classified as one group while the points above the line are classified as another group.

Use Cases

Supervised Learning has a large array of use cases:

  1. Analyzing customer reviews.
  2. Forecasting product sales.
  3. Face Recognition: Classification is used to classify if multiple images resemble the same person.
