Machine Learning

Written By Ved Panse Last Modified: 20 May 2022

Machine Learning is a popular buzzword in today's world. Let us try to understand what it means. Machine Learning (ML) means making machines make decisions by providing some experience. Many people confuse Artificial Intelligence (AI) and Machine Learning as a single concept. Machine Learning is a subset of AI, while AI covers other concepts like Deep Learning (DL).

Image from https://towardsdatascience.com/understanding-the-difference-between-ai-ml-and-dl-cceb63252a6c

Machine Experience

Suppose you want to create an ML program that categorizes images into two classes: cats and dogs. You will have to start by feeding the program some pictures of cats and dogs. You would also need to provide the program with the respective class of animals for those images (label each image as a cat or a dog). Thus, the machine would get a brief idea of the appearance of cats and dogs. The pictures (or the data) fed to the computer are called 'training data.' The training data is the experience that makes Machine Learning possible.

Types of Machine Learning

Machine Learning can be categorized into three types:

  1. Supervised Learning
  2. Unsupervised Learning
  3. Reinforcement Learning

Supervised Learning

Supervised learning is one of the most basic types of Machine Learning. It simply takes inspiration from the past data and predicts the future data. Such data is called 'labeled data.' This type of ML algorithm is most evident in a stock market website. The historical stock values are studied, and predictions are made.

In other words, the machine is told what to look for. It does so till it recognizes patterns (like periodicity) in the data up to accuracy. These patterns then help to make a prediction.

The "cats and dogs" example discussed above is an example of Supervised Learning.

Unsupervised Learning

Unsupervised Learning is more advanced than Supervised Learning. It does not require any previous/historical data to make predictions. Thus, it works on unlabeled data. In this type of learning, the data is clustered into groups. You can read more about this on this link. This type of learning does not need human supervision.

Reinforcement Learning

Reinforcement learning refers to making the machines learn to make an on-the-spot decision. It takes inspiration from the human way of thinking. This algorithm works in situations where neither the user nor the algorithm knows what will happen next. The program has to build its own experience from scratch and use it as and when needed.

For example, self-driving cars do not know what is coming next: will there be a pothole, a speed breaker, or a cross-road. Nor does the passenger know. For such situations, algorithms use reinforcement learning. In some games, the player's decision affects what will happen next. Such games use reinforcement learning.

How does the algorithm work?

The algorithm has a score. The score increases if the decision it produces is valid and appreciable by the client. On the other hand, the score decreases if the decision is either invalid or not appreciated. The algorithm studies this score and makes recommendations accordingly.

Use Cases

Machine Learning is now ubiquitous. Everything - right from voice assistants to self-driving car systems - uses machine learning.
