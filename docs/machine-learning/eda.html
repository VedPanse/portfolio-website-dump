What is EDA?

Exploratory Data Analysis

Written By Ved Panse Last Modified: 20 May 2022

Before diving straight into Machine Learning, we must study Exploratory Data Analysis (EDA). EDA is the analysis of given data to predict future data. It is nothing other than the analytical comprehension taught in schools.

What is EDA?

The data presented is converted into a graph form by using various tools (visit this link to my Plotly tutorial to know how). Then, it is the person's task to look at the graphs and analyze the trends (and the factors that influence them) to create an analytical report. I had written a technical blog on EDA, teaching how you can write one.

Since EDA would be hectic if the data is too voluminous, machine learning is used.
